During a residency for creating a dance piece in the EinTanzHaus (Mannheim), in less than three weeks of experimentation i crafted a device able to track the position of 7 dancers in an 8x8 meter space by using openCV. The device was running on a raspberry pi 3 with a 160 degree infrared camera and sending message to a client running on desktop that parsed and encapsulated all the data in a easy to use c++ class. You can find the code and hardware notes for client and server (and other similar tools) here.
